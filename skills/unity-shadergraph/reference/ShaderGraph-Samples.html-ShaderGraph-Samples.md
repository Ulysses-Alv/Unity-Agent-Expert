# Shader Graph samples

The Shader Graph package offers many different samples that demonstrate how to use Shader Graph features in various contexts.

| Topic | Description |
| --- | --- |
| [Import Shader Graph samples](ShaderGraph-Samples-Import.html) | Learn how to import the Shader Graph samples to your project. |
| [Procedural Patterns sample](Shader-Graph-Sample-Procedural-Patterns.html) | Learn how to generate patterns procedurally with Shader Graph. |
| [Node Reference sample](Shader-Graph-Sample-Node-Reference.html) | Learn in context about the nodes from the Shader Graph [node library](Node-Library.html). |
| [Feature Examples sample](Shader-Graph-Sample-Feature-Examples.html) | Learn about shader techniques such as angle blending, triplanar projection, parallax mapping, and custom lighting. |
| [Production Ready Shaders sample](Shader-Graph-Sample-Production-Ready.html) | Learn how to create production ready shader graphs that emulate terrain vegetation, rock details, water effects, weather effects, and other post-processing effects. |
| [UGUI Shaders sample](Shader-Graph-Sample-UGUI-Shaders.html) | Learn how to build user interface elements with Shader Graph. |
| [Custom Material Property Drawers sample](Shader-Graph-Sample-Custom-Material-Property-Drawers.html) | Learn how to set up a shader graph to display a property in the form of a min/max slider and add a help box in the material's Inspector UI. |
| [Custom Lighting sample](Shader-Graph-Sample-Custom-Lighting.html) | Learn how to create custom lighting models in Shader Graph. |
| [Terrain Shaders sample](Shader-Graph-Sample-Terrain.html) | Learn how to build terrain shaders with Shader Graph. |
