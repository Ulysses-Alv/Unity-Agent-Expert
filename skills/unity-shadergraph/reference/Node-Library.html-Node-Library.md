# Node library

Explore nodes that enable color and channel manipulation, mathematical and procedural generation, input data handling, custom texture management, UV mapping, utility logic, and shader data representation.

## Graph nodes

| **Topic** | **Description** |
| --- | --- |
| [Artistic](Artistic-Nodes.html) | Learn about color adjustment, blending, filtering, masking, normal map manipulation, and color space conversion. |
| [Channel](Channel-Nodes.html) | Learn about combining, splitting, reordering, or flipping vector and color channels. |
| [Custom Render Texture nodes](Custom-Render-Texture-Nodes.html) | Learn about properties and data of custom render textures. |
| [Input](Input-Nodes.html) | Learn about values, mesh attributes, gradients, matrices, deformation data, PBR parameters, scene information, and texture sampling options. |
| [Math](Math-Nodes.html) | Learn about mathematical operations. |
| [Procedural](Procedural-Nodes.html) | Learn about creating patterns, noise textures, and geometric shapes. |
| [SpeedTree](SpeedTree8-SubGraphAssets.html) | Learn about using SpeedTree 8 Sub Graph assets |
| [UI](ui-nodes.html) | Learn about nodes specifically designed for UI elements, including render type handling, element texture sampling, and layout-based UVs. |
| [Utility](Utility-Nodes.html) | Learn about basic preview, sub-graph referencing, and essential logic operations. |
| [UV](UV-Nodes.html) | Learn about manipulation and mapping effects, enabling advanced texture animations, coordinate transformations, and warping techniques. |
