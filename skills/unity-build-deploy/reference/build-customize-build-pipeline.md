# Customize the build pipeline

Unity’s build pipeline transforms your project into Player builds for target platforms. You can customize this process using custom **scripts**, callbacks, and command-line tools to automate builds and integrate with your development workflow.

| **[Create a build from the command line](build-command-line.md)** | Build Unity players from the command line without opening the Editor. |
| **[Use build callbacks](build-callbacks.md)** | Implement build callbacks to insert custom behavior into the Player build process. |
| **[Platform build path reference](build-path-requirements.md)** | Reference for build file extensions and path requirements for different platforms. |
