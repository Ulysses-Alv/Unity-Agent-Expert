# Sysroot Linux Arm64

[com.unity.sdk.linux-arm64](https://docs.unity3d.com/Packages/com.unity.sdk.linux-arm64@1.0/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

This package contains the sysroot used to build and run Unity players on Arm64 Linux systems. It includes the system libraries and headers required by Unity’s runtime environment to ensure compatibility with common Linux distributions during both compilation and execution.
