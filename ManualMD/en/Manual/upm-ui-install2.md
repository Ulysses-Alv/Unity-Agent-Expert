# Install a UPM package from the Asset Store

Use this procedure to install **UPM packages** that come from the **Asset Store**.

For information about installing UPM packages from a registry, refer to [Install a UPM package from a registry](upm-ui-install.md).

For information about adding **asset packages** (`.unitypackage` format) that come from the Asset Store, refer to [Download and import an asset package](upm-ui-import.md).

For information about the differences between UPM packages and asset packages, refer to [Differences between package types](upm-package-types.md).

## Prerequisites

Before you begin:

* Make sure you’ve acquired the package from the Asset Store. Refer to [Purchase or download a package from the Asset Store](AssetPackagesPurchase.md).
* Make sure the package appears in the [List panel](upm-ui-list.md) of the Package Manager window when you select the **My Assets** context.

## Procedure

1. Open the **Package Manager** window and select **My Assets** from the [navigation panel](upm-ui-nav.md).

   ![Select My Assets](../uploads/Main/upm-ui-myassets.png)

   Select **My Assets**
2. Select the package you want to install from the [list of packages](upm-ui-list.md). The package information appears in the [details panel](upm-ui-details.md).
3. Optional: If multiple versions are available, select the version to install. For more information on available versions, refer to [Find a specific version of a package](upm-ui-find-ver.md).
4. Select **Install**.

When the progress bar finishes, the new package is ready to use.

**Note**: You can install multiple packages with one click by using the multiple select feature. For more information, refer to [Perform an action on multiple packages or feature sets](upm-ui-multi.md).
