# Collecting performance data

Connect your application to the Profiler to measure its performance.

The way you connect to the Profiler depends on the platform you want to profile your application on.

| **Topic** | **Description** |

| **[Collect performance data in Play mode](profiling-play-mode.md)** | Profile in Play mode in the Unity Editor. |
| **[Collect performance data about the Unity Editor](profiling-edit-mode.md)** | Profile the Editor process. |
| **[Build Profiles Profile settings reference](profiler-build-settings-reference.md)** | Adjust how the Profiler connects to your application when you build it. |
