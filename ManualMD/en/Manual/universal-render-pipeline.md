# Using the Universal Render Pipeline

![Universal Render Pipeline cross-platform rendering example.](../uploads/Main/urp-example.png)

Universal Render Pipeline cross-platform rendering example.

The Universal Render Pipeline (URP) is a prebuilt [Scriptable Render Pipeline](scriptable-render-pipeline-introduction.md), made by Unity. URP provides artist-friendly workflows that let you quickly and easily create optimized graphics across a range of platforms, from mobile to high-end consoles and PCs.
