# Profiler modules

The Profiler window contains modules which display different types of data. You can select from default modules, or create your own custom modules to display data specific to your application.

| **[CPU Usage Profiler module](profiler-cpu.md)** | Collect data about the CPU timings of your code. |
| **[Memory Profiler module](profiler-memory.md)** | Collect data about memory usage in your application. |
