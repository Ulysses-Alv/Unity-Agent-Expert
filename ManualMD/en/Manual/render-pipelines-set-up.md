# Set up a render pipeline

To create a new project with one of the **render pipelines**, use the [Unity Hub](https://docs.unity3d.com/hub/manual/). Refer to [Manage your templates](https://docs.unity3d.com/hub/manual/project-create.html) for more information.

To convert a project from the Built-In Render Pipeline to URP or HDRP, refer to the following:

* [URP Render Pipeline Converter](urp/features/rp-converter.md)
* [HDRP Pipeline Wizard](https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@17.0/manual/Render-Pipeline-Wizard.html)

After you choose a rendering pipeline, you can [Choose a lighting setup](choose-a-lighting-setup.md).
