# Editor Coroutines

[com.unity.editorcoroutines](https://docs.unity3d.com/Packages/com.unity.editorcoroutines@1.0/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

The editor coroutines package allows developers to start constructs similar to Unity’s monobehaviour based coroutines within the editor using abitrary objects.

## Version information
