# Optimize mesh rendering using level of detail (LOD)

Strategies and resources for applying **level of detail** (LOD) techniques to meshes to reduce rendering workload.

This section contains information on LOD techniques for meshes. For information on LOD techniques for **shaders**, refer to [SubShader LOD value](SL-ShaderLOD.md).

| **Topic** | **Description** |
| --- | --- |
| [Introduction to level of detail](LevelOfDetail.md) | Understand level of detail (LOD) and LOD indices, and learn about the LOD features available in Unity. |
| [Mesh LOD](lod/mesh-lod-landing.md) | **Mesh** LOD is a Unity feature that reduces the number of polygons to draw with minimum memory footprint and computational overhead. |
| [LOD Group](lod/lod-group-landing.md) | LOD Group is a flexible Unity LOD solution. It provides more optimization options than the [Mesh LOD](lod/mesh-lod-landing.md) feature, but has a larger memory footprint and computational overhead. |
| [2D images for low level of detail (LOD)](2d-images-lod.md) | Techniques and resources for using 2D **billboards** to simplify 3D meshes that are far away. |
