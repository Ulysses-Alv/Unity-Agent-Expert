# Select tiles with the Select tool

Use the **Select** tool to select tiles on the **tilemap** by selecting the cell the tile is on. You can also click and drag the tool over several cells to select them all at once.

The [Grid Selection properties](./grid-selection-properties-reference.md) become available in the ****Inspector**** window for the cells you select with the **Select** tool. You can further change cell contents with the **gizmos** and options available in the [Modify Tilemap section](./modify-tilemap-reference.md).

1. Select the **Select** tool from the toolbar, or press its shortcut key **S**.  
   ![The Tile Palette window with the mouse cursor over the Select tool button.](../../../../..//uploads/Main/2d-palette-toolbar-select-hl.png)
2. Select a tile (highlighted in orange) on the tilemap with the **Select** tool.  
   ![The Scene view with an example of tile selected in the tilemap, and its properties in the Inspector.](../../../../../uploads/Main/2d-palette-select-tile.png)

When you select tiles with the **Select** tool, more properties and options become available for the selected cells in the [Grid Selection Properties](./grid-selection-properties-reference.md) and [Modify Tilemap](./modify-tilemap-reference.md) sections. Refer to their respective documentation to learn how you can adjust their properties and options.
