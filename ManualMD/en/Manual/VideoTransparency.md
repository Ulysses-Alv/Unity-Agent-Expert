# Video transparency support

Unity’s [Video Clip](class-VideoClip.md) and [Video Player](class-VideoPlayer.md) components support alpha values in video files, also known as [transparency](StandardShaderMaterialParameterAlbedoColor.md).

This information covers the different types of transparent videos and how to set them up for use in Unity.
