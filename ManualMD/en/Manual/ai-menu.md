# AI menu reference

Use the **AI** menu to track points usage, manage your account, and launch the features.

![AI window displaying the points available](../uploads/Main/ai-window.png)

AI window displaying the points available

| Options | Description |
| --- | --- |
| **Points** | Displays your available AI points. Using various Unity AI features consumes points, and the points counter updates to reflect your current point balance. Your point balance number includes both monthly allocated points (if you’re a Unity paid user) and any additional points you have purchased separately.   **Note**: Points are free during beta. |
| **See what’s New** | Provides onboarding content, tutorials, and updates on Assistant, Generators, and Sentis. |
| **View Documentation** | Opens the Unity AI documentation. |
| **Manage Account** | Opens Unity Cloud account settings for [AI-related services](https://cloud.unity.com/home/ai/settings). |
| **Check for Updates** | Links to the Package Manager to check for updates to Unity AI packages. |
| **Open Assistant** | Opens Unity Assistant. |
| **Generate New** | Opens asset generation options for Animation, Material, Sound, **Sprite**, and Texture. |
