# Visualize shadow cascades

To make configuration easier, use the [Rendering Debugger](features/rendering-debugger.md) to visualize shadow cascades. In the **Lighting** section of the **Rendering Debugger** window, set **Lighting Debug Mode** to **Shadow Cascades**.

![Rendering Debugger > Lighting > Lighting Debug Mode > Shadow Cascades](../../uploads/urp/shadows/rendering-debugger-shadow-cascades.png)
