# Create a Universal Render Pipeline asset

Create a [Universal Render Pipeline Asset](universalrp-asset.md) to configure global rendering and quality settings and instantiate the URP pipeline for your project.

To create a URP Asset:

1. Select a target folder in the **Project** window.
2. Create a Universal Render Pipeline Asset (menu: **Assets** > **Create** > **Rendering** > **URP Asset (with Universal Renderer)**).

Alternatively, right-click in the **Project** window, and select **Create** > **Rendering** > **URP Asset (with Universal Renderer)**.

Unity creates a URP Asset in the selected folder.
