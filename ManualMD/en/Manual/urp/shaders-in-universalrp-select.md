# Assign a shader to a material in URP

To select and use a **Shader**:

1. In your Project, create or find the Material you want to use the Shader on. Select the **Material**. A Material **Inspector** window opens.
2. Click **Shader**, and select **Universal **Render Pipeline****, then the shader you want to use.
