# Shader graph material Inspector window reference for URP

Explore the properties and behaviors of the default shader graph materials in the Universal Render Pipeline (URP), in the **Assets** > **Create** > **Shader Graph** > **URP** menu.

For more information about **shader** graph, refer to [Shader Graph](https://docs.unity3d.com/Packages/com.unity.shadergraph@latest).

| **Topic** | **Description** |
| --- | --- |

| **[Unlit shader graph reference](prebuilt-shader-graphs-urp-unlit.md)** | Explore the properties and settings you can use to customize the default Unlit shader graph. |
| **[Sprite Custom Lit shader graph reference](prebuilt-shader-graphs-urp-sprite-custom-lit.md)** | Explore the properties and settings you can use to provide custom lighting for 2D **sprites**. |
| **[Sprite Unlit shader graph reference](prebuilt-shader-graphs-urp-sprite-unlit.md)** | Explore the properties and settings of the default shader graph for unlit 2D sprites. |
| **[Sprite Lit shader graph reference](prebuilt-shader-graphs-urp-sprite-lit.md)** | Explore the properties and settings of the default shader graph for lit 2D sprites. |
| **[Decal shader graph reference](prebuilt-shader-graphs-urp-decal.md)** | Explore the properties and settings you can use to customize the default Decal shader graph. |
| **[Terrain Lit shader graph reference](prebuilt-shader-graphs-urp-terrain-lit.md)** | Explore the properties and settings you can use to customize the default **Terrain** Lit shader graph. |
| **[Canvas shader graph reference](prebuilt-shader-graphs-urp-canvas.md)** | Explore the properties and settings you can use to apply Canvas shader graph materials to user interface elements you create using uGUI (Unity UI). |
| **[Fullscreen shader graph reference](prebuilt-shader-graphs-urp-fullscreen.md)** | Explore the properties and settings you can use to customize the default Fullscreen shader graph. |
