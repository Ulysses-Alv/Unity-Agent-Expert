# Asset Store package revenue and sales

When a customer buys a package on the **Asset Store**, information about that sale appears in the Asset Store publisher portal. Learn how to manage revenue from sales of your Asset Store Packages and create a free customer voucher.

| **Topic** | **Description** |
| --- | --- |
| [Collect revenue from Asset Store package sales](AssetStorePayouts.md) | Learn how to collect customer payments for your packages on the Asset Store. |
| [Refund customers](AssetStoreRefunding.md) | Learn how to verify a refund request and refund a customer for an Asset Store package sale. |
| [Issue an Asset Store voucher](AssetStoreVouchers.md) | Learn how to create a voucher code to give a customer one of your Asset Store package for free. |
| [Offer an Asset Store package at a discount](AssetStorePromotion.md) | Learn how to include a published package in an Asset Store discount period or offer it at a lower price at any time. |
