# Caching assets

If you’re working on a large project with lots of assets, you can use Unity **Accelerator** to reduce iteration time. Unity Accelerator uses the Asset Import Pipeline to coordinate asset sharing when your team is working on the same local network so that you don’t need to reimport portions of your project.
