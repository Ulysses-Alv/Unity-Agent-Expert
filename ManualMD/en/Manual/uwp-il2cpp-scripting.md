# IL2CPP scripting backend for UWP

IL2CPP is the only [scripting backend](scripting-backends.md) Unity supports to build Universal Windows Platform (UWP) applications. When you build a project using **IL2CPP**, Unity converts the Microsoft Intermediate Language code from your **scripts** into C++ before creating a native binary file. For more information on IL2CPP, refer to [IL2CPP overview](./scripting-backends-il2cpp.md).
