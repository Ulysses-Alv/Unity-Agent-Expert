# Change existing height with effects

**Note:** These tools are only available in the [Terrain Tools package](https://docs.unity3d.com/Packages/com.unity.terrain-tools@latest/).

The height effect tools change existing height features.

| Tool | Description |
| --- | --- |
| [Flatten terrain features](terrain-Effects-Flatten.md) | Flatten fine details on the **terrain** without changing the slope. |
| [Change peak sharpness](terrain-Effects-Sharpen.md) | Sharpen existing peaks and flatten the areas around them. |
| [Add contrast to features](terrain-Effects-Contrast.md) | Raise high areas and lower low areas at the same time. |
