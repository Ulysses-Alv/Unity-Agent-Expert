# Enable the SRP Batcher in URP

To activate the SRP Batcher in URP:

1. In the **Project window**, select the [URP Asset](urp/universalrp-asset.md).
2. In the **Inspector** for the URP asset, in the **Rendering** section, enable **SRP Batcher**. If this option is not visible, configure Unity to [display advanced properties in a URP Asset](urp/urp-asset-additional-settings.md).

> **Note**: If assets or shaders in a project are not optimized for use with the SRP Batcher, low performance devices might be more performant when you disable the SRP Batcher.
