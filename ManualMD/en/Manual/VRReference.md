# XR API reference

Understand the common APIs for **XR** development that Unity provides.

Unity provides APIs for XR development under the `XR` namespace. Refer to the following table to understand the XR APIs:

| **Topic** | **Description** |
| --- | --- |
| [XR.XRSettings](../ScriptReference/XR.XRSettings.md) | APIs for Global XR related settings. |
| [XR.XRDevice](../ScriptReference/XR.XRDevice.md) | APIs related to XR devices. |
| [XR.InputTracking](../ScriptReference/XR.InputTracking.md) | APIs for accessing XR input devices. |
