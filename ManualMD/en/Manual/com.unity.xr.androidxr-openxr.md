# Unity OpenXR Android XR

[com.unity.xr.androidxr-openxr](https://docs.unity3d.com/Packages/com.unity.xr.androidxr-openxr@1.2/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Enables you to build mixed reality apps for Android XR devices by implementing support for Android XR extensions.
