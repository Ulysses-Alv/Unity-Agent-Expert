# Types of assets

Unity supports a variety of asset types that you can import and use in your project. You can import assets from a variety of sources, and manage them in the Unity Editor. You can also group together configurations of assets into prefabs to reuse across your project.

| **Topic** | **Description** |
| --- | --- |
| **[Supported asset type reference](assets-supported-types.md)** | Reference of asset file formats that Unity can import in your projects. |
| **[Models](models.md)** | Import and configure 3D models from external modeling software for use in your **scenes**. |
| **[Meshes](mesh.md)** | Work with **mesh** data that defines the shape and structure of **3D objects** in Unity. |
| **[Prefabs](Prefabs.md)** | Create reusable **GameObject** templates that you can instantiate multiple times throughout your project. |
| **[Text assets](class-TextAsset.md)** | Import and use text files, JSON, XML, and other text-based data in your project. |
