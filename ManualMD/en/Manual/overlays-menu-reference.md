# Overlays menu reference

Use the overlays menu to choose which overlays to display or hide, and to create and load overlay presests.

You can set the overlays menu to the **toolbar** in panel or horizontal layout.
To learn more, refer to [Change the appearance and position of an overlay](overlays-position-and-display.md).

In the overlay menu panel layout you can use the drop-down to choose an overlay preset. To learn more, refer to [Create and manage overlay presets](manage-overlay-configurations.md).

The overlays menu contains the following properties.

| **Overlay** | **Icon** | **Description** |
| --- | --- | --- |
| **Tools** |  | Select a tool context, use [Transform](PositioningGameObjects.md) tools, and, if they’re available, component tools. | |
| **Tool Contexts** |  | Select a tool context. To learn more, refer to [Scene view context menu](SceneViewContextMenu.md). |
| **Tool Settings** |  | Control how the tool handle behaves. |
| **Grid and Snap** |  | Control the appearance and behavior of the **Scene** grid. To learn more, refer to [Grid and Snap toolbar overlay reference](overlay-grid-snap-reference.md). |
| **Draw Modes** |  | To learn more, refer to [Draw Modes overlay reference](overlays-reference-draw-modes.md). |
| **View Options** |  | To learn more about **Gizmos**, refer to the [Gizmos menu](GizmosMenu.md). |
| **Search** |  | Use Unity Search to find items in the Unity Editor. To learn more, refer to [Unity Search](search-overview.md). |
| **Orientation** |  | Change the Scene camera’s viewing angle and projection mode. To learn more, refer to [Orientation overlay](overlay-orientation.md). |
| **Cameras** |  | To learn more, refer to the [Cameras](cameras-overlay.md) overlay. To learn more about the **Scene view** camera, refer to the [Scene view Camera](SceneViewCamera.md). |
