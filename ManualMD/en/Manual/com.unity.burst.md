# Burst

[com.unity.burst](https://docs.unity3d.com/Packages/com.unity.burst@1.8/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Burst is a compiler that translates from IL/.NET bytecode to highly optimized native code using LLVM.
