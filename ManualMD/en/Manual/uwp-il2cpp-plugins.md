# Use UWP plug-ins with IL2CPP

You can use plug-ins to call code created outside of Unity from your C# **scripts**. The plug-in model for Universal Windows Platform (UWP) is similar to other Unity-supported platforms. For more information about plug-ins with Unity, refer to [Plug-ins](./plug-ins.md).
