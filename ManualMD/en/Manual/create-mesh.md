# Create a mesh

You can add meshes to a Unity project in the following ways:

* Create a **mesh** as part of a [model](models.md) in a 3D modeling program, then import it. Unity creates a mesh asset as a part of the model hierarchy when you import a model. For more information, see [Importing Models](ImportingModelFiles.md).
* Create a mesh directly with in-Editor tools. Unity has built-in primitive shapes such as spheres and capsules that you can place directly into a **Scene**. You can also use a package like [ProBuilder](https://docs.unity3d.com/Packages/com.unity.probuilder@latest) to model a custom mesh in the Editor.
* [Create meshes with code](creating-meshes.md), either at runtime or in the Editor.
