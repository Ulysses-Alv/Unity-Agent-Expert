# Feature sets

Feature sets are groups of [packages](PackagesList.md) that enable you to achieve specific results in Unity. For example, the 2D **feature set** provides all the functionality you need to create 2D games for various art styles.

All the packages in a feature set are compatible with each other for that version of the feature set. You can install feature sets using the [Package Manager](fs-install.md).

The following feature sets are available in this release of Unity:

* [2D](2DFeature.md): Create 2D game worlds with animated characters and 2D physics.
* [3D Characters and Animation](CharacterAnimationFeature.md): Animate characters and create cutscenes, sequences or movies.
* [3D World Building](WorldBuildingFeature.md): Use artist-friendly tools to create 3D, open-world environments that are believable and interactive.
* [AR](ARFeature.md): Create **augmented reality** experiences that interact with the real world.
* [ECS](ECSFeature.md): Create experiences using a data-oriented design approach, to scale processing in a highly performant manner.
* [Engineering](DeveloperToolsFeature.md): Enable support for integrated development environments, and use tools to test and analyze your project’s performance.
* [Gameplay and Storytelling](GameplayStorytellingFeature.md): Create dynamic **camera** shots, and arrange cinematic content to create a story for your project.
* [Mobile](MobileFeature.md): Optimize your apps for mobile devices.
* [VR](VRFeature.md): Add interactivity to your applications to create **virtual reality** experiences.
