# Offline documentation

You can download the Unity Documentation as a zip file for offline use. To download the current version of the Unity Documentation, click the following link.

[Download: Offline Unity Documentation](https://cloudmedia-docs.unity3d.com/docscloudstorage/en/6000.3/UnityDocumentation.zip) (Size: about 300MB)

The zip file contains the most up-to-date version of the **Unity Manual** and **Unity Scripting API**.
