# Programming with mathematics

Unity exposes several public APIs that allow you to work with mathematical concepts in your code. These APIs are useful for manipulating vectors, matrices, and other mathematical constructs that are common in game development.

| **Topic** | **Description** |
| --- | --- |
| **[Moving objects with vectors](scripting-vectors.md)** | Perform vector arithmetic with Unity’s 2D, 3D, and 4D vector classes to manage the position and speed of objects and the distance between them. |
| **[Using common math functions](class-Mathf.md)** | Use Unity’s [`Mathf`](../ScriptReference/Mathf.md) class to apply trigonometric, logarithmic, and other math functions commonly required in game development. |
| **[Using randomness](class-random.md)** | Use the [`Random`](../ScriptReference/Random.md) class to generate various types of random values commonly required in game development. |
