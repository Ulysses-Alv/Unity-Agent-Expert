# Input System

[com.unity.inputsystem](https://docs.unity3d.com/Packages/com.unity.inputsystem@1.19/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

A new input system which can be used as a more extensible and customizable alternative to Unity’s classic input system in UnityEngine.Input.
