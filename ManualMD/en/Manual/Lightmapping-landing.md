# Precalculating surface lighting with lightmaps

Resources for improving performance by setting up Unity to do complicated lighting calculations ahead of time, either by baking **lightmaps** into a build or creating lightmaps at runtime.

| **Page** | **Description** |
| --- | --- |
| [Global illumination](choose-a-lighting-setup.md) | Learn about baking lightmaps before runtime, or creating lightmaps at runtime using **Enlighten** Realtime Global Illumination. |
| [Baking lightmaps before runtime](Lightmapping-baking-before-runtime.md) | Resources for precalculating the lit colors of **GameObjects** and storing them in textures called lightmaps. |
| [Update lightmaps at runtime with Enlighten Realtime Global Illumination](realtime-gi-using-enlighten-landing.md) | Resources for using Enlighten to precalculate visibility information, which Unity uses to create and update lightmaps at runtime. |
| [Configuring lightmapping](configure-lightmapping-settings.md) | Resources for choosing between baked and real-time lightmapping, and changing how GameObjects contribute to lightmaps. |
