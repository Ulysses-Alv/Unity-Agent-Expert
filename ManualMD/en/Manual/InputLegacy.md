# Legacy Input

Unity has a built-in legacy input solution called the ****Input Manager****. This is not the recommended workflow, as it is less flexible than the Input System Package, and will be removed in future versions of Unity.

Unless you are supporting a legacy project that uses the older “Input Manager”, you should use the [Input System Package](Input.md).

The documentation for the legacy Input Manager remains in this section of the user manual to support legacy projects.

* [Legacy Input Manager](class-InputManager.md)
* [Legacy Mobile Input](MobileInput.md)
