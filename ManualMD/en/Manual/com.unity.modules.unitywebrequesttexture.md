# Unity Web Request Texture

[com.unity.modules.unitywebrequesttexture](https://docs.unity3d.com/6000.3/Documentation/ScriptReference/UnityEngine.UnityWebRequestTextureModule.html)

## Description

The UnityWebRequestTexture module provides the DownloadHandlerTexture class to use UnityWebRequest to download Textures.

## Version information

Built-in packages are fixed to a single version matching the Editor version.
