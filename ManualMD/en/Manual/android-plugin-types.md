# Android plug-in types

Unity supports multiple **plug-in** types for Android applications. Each plug-in type has its own capabilities so, if you want to create your own plug-in for a particular use-case, make sure to select the most relevant plug-in type for the use-case.

| **Topic** | **Description** |
| --- | --- |
| [Android Library and Android Archive plug-ins](AndroidAARPlugins.md) | Understand Android Library and Android Archive plug-ins, and how to use them to extend your application with C++ and Java code created outside of Unity. |
| [JAR plug-ins](AndroidJARPlugins.md) | Understand JAR plug-ins and learn how to use them to interact with the Android operating system or call Java code from C# **scripts**. |
| [Native plug-ins for Android](AndroidNativePlugins.md) | Understand how to use **native plug-ins** to call C/C++ code from C# scripts. |
| [Java and Kotlin source plug-ins](AndroidJavaSourcePlugins.md) | Understand how to use Java and Kotlin source code plug-ins to call Java or Kotlin code from C# scripts. |
