# Audio Source

[Switch to Scripting](../ScriptReference/AudioSource.md "Go to AudioSource page in the Scripting Reference")

The **Audio Source** is a component that plays an [Audio Clip](class-AudioClip.md) in your **scene** and provides options to customize its playback behavior.

This information covers how to set up an audio source and what each of the component settings do.
