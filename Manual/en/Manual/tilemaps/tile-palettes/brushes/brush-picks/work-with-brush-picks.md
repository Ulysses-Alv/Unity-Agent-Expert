# Work with Brush Picks

The following steps explain how to save and load a Brush Pick, and edit or remove a Brush Pick.

## Save a Brush Pick

![If there are no saved Brush Picks, the overlay displays instructions on how to save a Brush Pick.](../../../../../uploads/Main/tile-palette-brush-pick-empty.png)

If there are no saved Brush Picks, the overlay displays instructions on how to save a Brush Pick.

To save a Brush Pick:

1. Activate the [Pick Tool](../../tools/create-brush-picks-from-tiles-pick-tool.md) in the [Tile Palette](../../create-tile-palette.md) window.
2. Pick tiles with the [active brush](../active-brush.md) from the **Tile Palette** window or the ****Scene**** view. You can also adjust the brush’s settings in the [Brush Inspector window](../brush-inspector-reference.md).
3. Select the [Save button](./brush-picks-overlay-reference.md) in the Brush Pick overlay to save the picked tiles and brush settings as a Brush Pick.

## Edit a Brush Pick

To rename a Brush Pick:

1. Double-click the Brush Pick to edit the Brush Pick’s name.

To remove a Brush Pick:

1. Select the Brush Pick you want to remove.
2. Press Del (macOS: Cmd+Del) to remove the selected Brush Pick.

## Load a Brush Pick

To load a Brush Pick:

1. Select its thumbnail in the [overlay](./brush-picks-overlay-reference.md). Selecting a Brush Pick automatically switches your [active brush](../brush-inspector-reference.md) to match the Brush Pick’s **Brush** type, and load its stored brush settings.
2. Loading the Brush Pick automatically activates the [Paint tool](../../tools/paint-tiles-with-paint-tool.md), allowing you to paint with the Brush Pick straight away.
