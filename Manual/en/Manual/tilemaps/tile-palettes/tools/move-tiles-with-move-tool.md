# Move tiles with the Move tool

Use the **Move** tool to move the [Select tool](select-tool/select-tool-landing.md)’s tile selection to a position on the active **tilemap** while it is in Edit mode. The **Move** tool can’t select tiles to move by itself.

1. Select the tile or tiles that you want to move from the active [Tile Palette](../create-tile-palette.md) with the **Select** tool.
2. Switch to the **Move** tool by selecting it on the **Tile Palette** editor’s **toolbar**, or press its shortcut key **M**.

   ![The Tile Palette window with the mouse cursor over the Move tool button.](../../../../uploads/Main/2d-palette-toolbar-move-hl.png)

   The Tile Palette window with the mouse cursor over the Move tool button.
3. Click and drag the selected tiles (highlighted in orange) with the **Move** tool to a new position on the tilemap.

   ![The Scene view with an example of tiles moved on the tilemap.](../../../../uploads/Main/2d-palette-move-tile.png)

   The Scene view with an example of tiles moved on the tilemap.
