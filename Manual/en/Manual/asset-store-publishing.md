# Publishing Asset Store packages

You can publish packages to the **Asset Store** to share and monetize assets, tools, and extensions. You can publish packages in the **asset package** (`.unitypackage` files) format, or apply to publish in the Unity Package Manager (UPM) beta program.

| **Topic** | **Description** |
| --- | --- |
| **[Set up an Asset Store publisher profile](AssetStoreCreateAcct.md)** | Create a Unity ID and a publisher profile that you use to manage and publish Asset Store packages. |
| **[Add a user to a publisher profile](AssetStoreUsers.md)** | Allow access to your publisher portal account from multiple email addresses. |
| **[Publish an asset package on the Asset Store](AssetStorePublishing.md)** | Publish a package to the Asset Store in the asset package (`.unitypackage`) format. |
| **[Publish a UPM package on the Asset Store (Beta)](asset-store-upm.md)** | Publish a package to the Asset Store in the Unity Package Manager format. This is a beta workflow, only available to certain developers. |
