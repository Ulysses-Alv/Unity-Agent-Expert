# Extending the Editor with IMGUI

**Note**: It’s strongly recommended to use the [UI Toolkit](UIElements.md) to extend the [Unity Editor](UIE-support-for-editor-ui.md), as it provides a more modern, flexible, and scalable solution than IMGUI.

Unity lets you extend the editor with your own custom inspectors and **Editor Windows** and you can define how properties are displayed in the **inspector** with custom **Property Drawers**. This section explains how to use these features.

| Topic | Description |
| --- | --- |
| [Create custom Editor Windows with IMGUI](editor-EditorWindows.md) | Create custom windows in your app. |
| [Create custom Editors with IMGUI](editor-CustomEditors.md) | Create custom editors for components you commonly use. |
| [Use Property Drawers with IMGUI to customize the Inspector](editor-PropertyDrawers.md) | Customize the look of certain controls in the Inspector window. |
| [Create TreeView with IMGUI](TreeViewAPI.md) | Create custom tree views in the Editor. |
