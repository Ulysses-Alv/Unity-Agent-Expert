# 2D game development in URP

Create unique visual effects for your 2D URP project.

Use the different 2D URP features available to create the graphics and visual style you want in your 2D project. Simulate lighting in 2D **scenes** with [2D lighting in URP](urp/2d-index.md), or create a pixelated visual style with the [2D Pixel Perfect Camera](urp/2d-pixelperfect.md). You can use [3D GameObjects in 2D URP scenes](urp/2d-renderer-urp-features-landing.md) to create unique visual effects with 3D meshes in your 2D project.
