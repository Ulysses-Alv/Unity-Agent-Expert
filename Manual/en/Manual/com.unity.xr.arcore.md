# Google ARCore XR Plugin

[com.unity.xr.arcore](https://docs.unity3d.com/Packages/com.unity.xr.arcore@6.5/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Provides native Google ARCore integration for use with Unity’s multi-platform XR API.  
Supports the following features:  
-Efficient Background Rendering  
-Horizontal Planes  
-Depth Data  
-Anchors  
-Hit Testing  
-Occlusion
