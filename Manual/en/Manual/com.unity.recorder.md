# Recorder

[com.unity.recorder](https://docs.unity3d.com/Packages/com.unity.recorder@5.1/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

The Recorder allows you to capture and save data from the Unity Editor during Play mode: animations, videos, images, audio, AOVs, and more.
