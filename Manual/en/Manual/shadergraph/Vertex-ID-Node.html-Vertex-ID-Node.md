# Vertex ID Node

## Description

Provides access to the mesh vertex or fragment's **Vertex ID** value.

## Ports

| Name | Direction | Type | Binding | Description |
| --- | --- | --- | --- | --- |
| Out | Output | Float | None | **Vertex ID** for the Mesh Vertex/Fragment. |
