# Make LOD transitions smooth in Mesh LOD

To make **LOD** transitions smooth, enable LOD cross-fading. For more information, refer to [Introduction to level of detail](../LevelOfDetail.md).

Follow these steps:

1. Enable [GPU Resident Drawer](../urp/gpu-resident-drawer.md).
2. In the [Universal Render Pipeline (URP) Asset](../urp/universalrp-asset.md), enable **LOD Cross Fade**.
