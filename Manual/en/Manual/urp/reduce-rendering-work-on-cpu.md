# GPU Resident Drawer

To speed up rendering, enable the GPU Resident Drawer so Unity automatically uses the `BatchRendererGroup` API to reduce the number of draw calls.

| **Page** | **Description** |
| --- | --- |
| [Enable the GPU Resident Drawer](gpu-resident-drawer.md) | Automatically use the `BatchRendererGroup` API to use instancing and reduce the number of draw calls. |
| [Make a GameObject compatible with the GPU Resident Drawer](make-object-compatible-gpu-rendering.md) | Include or exclude a **GameObject** from the GPU Resident Drawer. |
