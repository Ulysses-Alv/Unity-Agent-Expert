# Introduction to lighting in the Universal Render Pipeline

All of Unity’s render pipelines share common lighting functionality, but each **render pipeline** has some important differences.

In URP, the following applies:

* The [Light component inspector](../light-component.md) displays some URP-specific controls.
* The [Universal Additional Light Data](../universal-additional-light-data.md) component allows Unity to store Light-related data that is specific to URP.
