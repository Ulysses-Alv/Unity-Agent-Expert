# Color Lookup Override reference for URP

The Color Lookup override lets you use a color lookup table (LUT) texture to change the color, contrast, and tone of the rendered image. This operation is often called color grading.

## Properties

| **Property** | **Description** |
| --- | --- |
| **Lookup Texture** | Contains a color lookup table (LUT) texture. |
| **Contribution** | Defines how much of the color-graded image Unity blends with the original image. |
