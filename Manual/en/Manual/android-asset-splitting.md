# Introduction to asset splitting

Depending on which publishing format your application uses, there is a different solution for asset splitting.

The following table shows which asset splitting solution is for which publishing format:

| **Publishing format** | **Asset splitting solution** |
| --- | --- |
| ****APK**** | [APK expansion files](android-OBBsupport.md) |
| [Android App Bundle](https://developer.android.com/guide/app-bundle) (AAB) | [Play Asset Delivery](play-asset-delivery.md) |

**Note**: Some [digital distribution services](android-distribution.md) require applications to use a specific publishing format. For example, [Google Play](https://play.google.com/about/howplayworks/) requires new applications to use the AAB format which means your application must use [Play Asset Delivery](play-asset-delivery.md) if you want to distribute your application through Google Play.
