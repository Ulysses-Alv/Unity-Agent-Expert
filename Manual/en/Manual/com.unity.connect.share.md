# WebGL Publisher

[com.unity.connect.share](https://docs.unity3d.com/Packages/com.unity.connect.share@4.2/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

This tool allows you to make a WebGL build of your current project and upload it to Unity’s website, where it can be shared with others.
