# Analyzing asset processes

Optimize and measure the performance of assets in your project. The Asset Loading Profiler module and the `AsyncReadManagerMetrics` API allow you to measure how quickly Unity loads the assets in your project. You can also use the Import Activity window to measure import timings.

| **Topic** | **Description** |
| --- | --- |
| **[Collect asset loading metrics](AsyncReadManagerMetrics.md)** | Monitor asset loading at runtime. |
| **[Asset Loading Profiler module reference](profiler-asset-loading-module.md)** | Visualize the number of read operations on the assets in your project. |
| **[Analyze the import process](import-analyze.md)** | Analyze and debug asset importers. |
| **[Analyze built assets](assets-analyze-built-assets.md)** | Use the WebExtract and binary2text tools to convert and view asset archive files. |
