# Primary sculpting tools

Primary tools for sculpting height are available in both the built-in **Terrain** system and the Terrain Tools package. The tools are:

| **Tool** | **Description** |
| --- | --- |
| [**Set the height of an area or tile**](terrain-SetHeight.md) | Set a single height for a whole tile or parts of it. |
| [**Sculpt the terrain with a brush**](terrain-RaiseLowerTerrain.md) | Sculpt areas of the terrain. |
| [**Paint holes in the terrains**](terrain-PaintHoles.md) | Paint holes in the terrain to allow characters and objects to enter caves or other underground structures. |
| [**Smooth the terrain’s height**](terrain-SmoothHeight.md) | Smooth sharp formations and transitions between areas of the terrain. |
| [**Stamp a shape onto the terrain**](terrain-StampTerrain.md) | Use a stamp or **mesh** to add or remove height as a set shape. |
