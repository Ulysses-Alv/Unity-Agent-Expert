# Default Scene view overlay reference

From the Overlays menu, you can [control which overlays are in the Scene view](display-and-hide-overlay.md) and [manage your overlay configurations](manage-overlay-configurations.md). To display the Overlays menu in the **Scene** view, press the ` key. To restore the default overlay configuration, select the **Default** overlay configuration from the Overlays menu.

By default, the following overlays display in the **Scene view**.

![Overlays](../uploads/Main/overlays-default-view.png)

Overlays

| **Overlay** | **Description** |
| --- | --- |
| A: **Tools** | Use the Tools overlay to select a tool context, use [Transform](PositioningGameObjects.md) tools, and, if they’re available, component tools. |
| B: **Tool Settings** | Use the Tool Settings overlay to display available settings for the tool you have selected. For example, if you select a [Transform tool](PositioningGameObjects.md), use the Tool Settings overlay to select the [location of any Transform tool Gizmo and the handles used to manipulate the Gizmo](PositioningGameObjects.md#GizmoHandlePositions). |
| C: **Grid and Snap** | Control how **GameObjects** snap to the grid. To learn more, refer to [Grid and Snap overlay](overlay-grid-snap-reference.md). |
| D: **Draw Modes** | Control the appearance of meshes in the Scene view. To learn more, refer to [Draw Modes overlay reference](overlays-reference-draw-modes.md). |
| E: **View Options** | Use the [View Options overlay](overlays-reference-view-options.md) to select view options, control lighting, and control audio for the Scene view. These controls only affect the Scene view and don’t affect the built game. |
| F: **Orientation** | Use the [Orientation overlay](overlay-orientation.md) to view the Scene **camera**’s current orientation and to change its viewing angle and projection mode. |
| G: **Overlay Menu** | Show or hide each overlay in the Scene view. To learn more, refer to [Overlays menu reference](overlays-menu-reference.md). |
