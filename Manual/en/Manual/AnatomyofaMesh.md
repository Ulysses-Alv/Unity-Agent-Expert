# Mesh data

Resources on the data that a **mesh** contains, and how to view it in the Unity Editor.

| **Topic** | **Description** |
| --- | --- |
| [Mesh vertex data](mesh-vertex-data.md) | Learn about mesh vertex data, which describes a collection of positions in 3D space. |
| [Mesh topology data](mesh-topology-data.md) | Learn about mesh topology data, which describes the type of face that a mesh has. |
| [Mesh index data](mesh-index-data.md) | Learn about mesh index data, which describes how vertices combine to create the mesh’s surface, based on the topology. |
| [Mesh data for deformable meshes](mesh-data-deformable-meshes.md) | Learn about data specific to deformable meshes. |
| [View mesh data visualizations](view-mesh-data-visualizations.md) | Use the Mesh asset Preview window to view and explore visualizations of a mesh and its data. |
