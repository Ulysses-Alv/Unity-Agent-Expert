# Create a custom search provider

You can use the Search API to create your own [search provider](search-filter-by-provider.md). A custom search provider can combine search results from different sources, exclude properties from the search, and perform actions on the search results.

This section provides information on how to create a custom search provider.

| Topic | Description |
| --- | --- |
| [The SearchProvider class](api-search-provider-class.md) | Execute a search with the SearchProvider class. |
| [Register a search provider](api-register-provider.md) | Register a search provider with the SearchService. |
| [Search and fetch](api-searching.md) | Add a search and fetch function to the search provider, and use a fuzzy search. |
| [Action handlers](api-action-handler.md) | Create action handlers for the search provider. |
