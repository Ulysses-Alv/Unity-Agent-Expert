# Unity OpenXR Meta

[com.unity.xr.meta-openxr](https://docs.unity3d.com/Packages/com.unity.xr.meta-openxr@2.4/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

Enables you to build mixed reality apps for Meta Quest devices by implementing support for Meta’s OpenXR extensions.
