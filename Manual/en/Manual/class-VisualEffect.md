# Visual Effect component

The Visual Effect component contains basic controls to configure an instance of a Visual Effect in a **Scene**.

This component only works when you install the Visual Effect Graph package. For information on how to install the Visual Effect Graph, see the [packages documentation](upm-ui-install.md).

For information on how to use the Visual Effect component, see the Visual Effect Graph [package documentation](https://docs.unity3d.com/Packages/com.unity.visualeffectgraph@latest/index.html?subfolder=/manual/VisualEffectComponent.html).
