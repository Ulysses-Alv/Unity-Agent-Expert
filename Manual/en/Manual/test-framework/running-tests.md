# Running tests

Run your Unity Test Framework tests from the Unity Editor’s ****Test Runner**** window, from code, or from the Editor command line.

| **Topic** | **Description** |
| --- | --- |
| **[Run tests from the Test Runner window](workflow-run-test.md)** | Use the Unity Editor’s built-in **Test Runner** window to run tests in Edit mode, Play mode, or in a standalone platform Player. |
| **[Run tests from the command line](run-tests-from-command-line.md)** | Run tests from the Editor command line with Unity Test Framework’s command-line arguments. |
| **[Run tests from code](running-tests-from-code.md)** | Run tests from code with the [TestRunnerAPI](https://docs.unity3d.com/Packages/com.unity.test-framework@latest/index.html?subfolder=/api/UnityEditor.TestTools.TestRunner.Api.TestRunnerApi.html). |
| **[Run Play mode tests in a Player](workflow-run-playmode-test-standalone.md)** | Build and run your tests on a standalone Player for your target platform. |
| **[Modifying a Player build for tests](reference-attribute-testplayerbuildmodifier.md)** | Modify the build options for a platform Player built to run tests on. |
