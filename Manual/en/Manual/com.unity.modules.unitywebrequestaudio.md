# Unity Web Request Audio

[com.unity.modules.unitywebrequestaudio](https://docs.unity3d.com/6000.3/Documentation/ScriptReference/UnityEngine.UnityWebRequestAudioModule.html)

## Description

The UnityWebRequestAudio module provides the DownloadHandlerAudioClip class to use UnityWebRequest to download AudioClips.

## Version information

Built-in packages are fixed to a single version matching the Editor version.
