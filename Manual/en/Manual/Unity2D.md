# 2D game development

You can use the Unity Editor to create projects in 3D and 2D. Read this section to learn how to develop 2D projects in the Editor, including how to use sprites, create Tilemaps, and use the 2D physics system.

![](../uploads/Main/2dGames.jpg)

Many areas of Unity’s documentation apply to both 2D and 3D development. This section focuses on 2D-specific features and functionality. To choose the right mode for your project, refer to [2D and 3D Projects](2Dor3D.md). For a comparison on two modes, refer to [2D and 3D mode settings](2DAnd3DModeSettings.md).
