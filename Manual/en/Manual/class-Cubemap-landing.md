# Cubemaps

Resources for creating and using a collection of six square textures that represent the reflections in an environment, and form an imaginary cube around a **GameObject**.

| [Create a cubemap](class-Cubemap-create.md) | Generate a cubemap from textures, or the contents of a **scene**. |
| [Create a cubemap array](class-CubemapArray-create.md) | Use a script to create a cubmap array. |
| [Sample a cubemap array in a shader](class-CubemapArray-use-in-shader.md) | Sample a cubemap array in High-level **Shader** Language (HLSL). |
