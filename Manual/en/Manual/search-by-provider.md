# Search provider references

Use search providers to focus your search.

This section provides a reference for individual search providers, grouped by their purpose. For information on toggling and using search providers, refer to [Use search providers](search-use-provider.md). For information about creating a search provider, refer to [Create a custom search provider](create-search-provider.md).

## Search your project and scenes

[Search your project and scenes](search-project-and-scenes-overview.md) with the Hierarchy, Project, and Asset Database search providers.

| **Provider** | **Token** | **Description** |
| --- | --- | --- |
| [Search GameObjects with the Hierarchy search provider](search-scene.md) | `h` | Search **GameObjects** in the loaded **scenes**. |
| [Search assets with the Project search provider](search-assets.md) | `p` | Search the project’s assets. |
| [Search the Asset Database](search-adb.md) | `adb` | Search the Asset Database. |
