# GameObject fundamentals

Understand the basic workflows and operations for working with GameObjects in Unity. Learn how to manipulate transforms, control **GameObject** visibility and activation, create primitive shapes for prototyping, and understand how Unity manages GameObjects internally.

| **[Transforms](class-Transform.md)** | Information about transforms for GameObjects in Unity. |
| **[Rotation and orientation](rotation-orientation.md)** | Use euler angles or **quaternions** in Unity’s left hand coordinate system to control the rotation and orientation of GameObjects. |
| **[Static GameObjects](StaticObjects.md)** | Information about static GameObjects. |
| **[Deactivating GameObjects](DeactivatingGameObjects.md)** | Learn about how to deactivate GameObjects. |
| **[Primitive and placeholder GameObjects](PrimitiveObjects.md)** | Create basic 3D shapes like cubes, spheres, and cylinders. |
| **[2D Primitive GameObjects](2DPrimitiveObjects.md)** | Create basic 2D shapes like **sprites**, circles, and polygons. |
