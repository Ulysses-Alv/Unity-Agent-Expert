# Multiplayer Services

[com.unity.services.multiplayer](https://docs.unity3d.com/Packages/com.unity.services.multiplayer@2.1/manual/index.html) ![](../uploads/Main/iconRel.png)

## Description

The Multiplayer Services package is a one-stop solution for adding multiplayer elements to a game developed in Unity. The Unity Gaming Services (UGS) Multiplayer services powers these capabilities to define how groups of players interact in your games.
