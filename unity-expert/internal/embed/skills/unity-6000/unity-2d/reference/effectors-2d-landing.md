# Effectors 2D

Use Effector 2D [components](../../UsingComponents.md) with [Collider 2D](../collider/collider-2d-landing.md) components to direct the forces of [physics](../../PhysicsSection.md) when [GameObject](../../GameObjects.md) **colliders** come into contact with each other. You can use the following Effectors 2D components in Unity:

| **Page** | **Description** |
| --- | --- |
| [Area Effector 2D reference](area-effector-2d-reference.md) | Explore the properties of the Area Effector 2D used to arbitrarily vary force and angle magnitude. |
| [Buoyancy Effector 2D reference](buoyancy-effector-2d-reference.md) | Explore the properties of the Buoyancy Effector 2D used to simulate buoyancy, fluid flow and fluid drag. |
| [Point Effector 2D reference](point-effector-2d-reference.md) | Explore the properties of the Point Effector 2D used to attract or repulse against a given source point. |
| [Platform Effector 2D reference](platform-effector-2d-reference.md) | Explore the properties of the Platform Effector 2D used to create platform behavior, such as one-way **collisions**. |
| [Surface Effector 2D reference](surface-effector-2d-reference.md) | Explore the properties of the Surface Effector 2D used to create conveyor belts. |
