# 2D physics

Unity’s physics system lets you handle 2D physics to make use of optimizations available with 2D.

**Note:** You can also use a separate `LowLevelPhysics2D` API to use 2D physics. The API doesn’t interact with or affect built-in Unity 2D physics components such as **RigidBody** 2D and **Collider** 2D. For more information, refer to [Introduction to the LowLevelPhysics2D API](../2d-physics-api/2d-physics-api-introduction.md).
