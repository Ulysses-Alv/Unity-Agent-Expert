# Sorting groups

**Sorting Groups** allow you to group **GameObjects** with [Sprite Renderers](../renderer/sprite-renderer-reference.md) together, and control the order in which they render their sprites. Unity renders **Sprite** Renderers in the same Sorting Group together, as if they are a single GameObject.

| [**Use Sorting Groups**](use-sorting-groups.md) | Control the rendering order of complex 2D multi-Sprite characters. |
| [**Sorting group reference**](sorting-group-reference.md) | Set the priority of a a sorting group in the rendering queue. |
