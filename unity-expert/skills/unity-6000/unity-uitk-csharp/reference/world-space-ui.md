# World Space UI

You can create a World Space UI and position it alongside 2D or **3D objects** in the **Scene**, rather than displaying them on the screen as with the **Screen Space Overlay** render mode.

| **Topic** | **Description** |
| --- | --- |
| [Create a World Space UI](create-world-space-ui.md) | Set the render mode of the Panel Settings asset to World Space to create a World Space UI. |
| [Panel Input Configuration properties](world-space-panel-input-configuration.md) | Configure the input settings for the World Space UI. |
| [World Space UI size and position examples](world-space-ui-size-and-position-examples.md) | Learn how to size and position World Space UI with practical examples. |
