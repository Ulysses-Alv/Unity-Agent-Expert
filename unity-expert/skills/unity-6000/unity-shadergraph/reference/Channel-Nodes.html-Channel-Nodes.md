# Channel nodes

Combine, split, reorder, or flip vector and color channels.

| **Topic** | **Description** |
| --- | --- |
| [Append](Append-Node.html) | Combine two float or vector inputs into a single new vector of variable dimensions. |
| [Combine](Combine-Node.html) | Creates new vectors from the four inputs R, G, B and A. |
| [Flip](Flip-Node.html) | Flips the individual channels of input In selected by the node's parameters. |
| [Split](Split-Node.html) | Splits the input vector In into four Float outputs R, G, B and A. |
| [Swizzle](Swizzle-Node.html) | Creates a new vector from the reordered elements of the input vector. |
