# Vertex Color Node

## Description

Provides access to the mesh vertex or fragment's **Vertex Color** value.

## Ports

| Name | Direction | Type | Binding | Description |
| --- | --- | --- | --- | --- |
| Out | Output | Vector 4 | None | **Vertex Color** for the Mesh Vertex/Fragment. |
